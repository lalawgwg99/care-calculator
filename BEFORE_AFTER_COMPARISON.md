## 🔄 如何切換

### 從舊版切換到新版
```typescript
// 1. 備份舊版
mv app/page.tsx app/page.old.tsx

// 2. 使用新版（已完成）
// app/page.tsx 已經是新版

// 3. 測試
npm run dev
```

### 從新版切換回舊版
```typescript
// 1. 恢復舊版
mv app/page.old.tsx app/page.tsx

// 2. 確保 CareCalculator.tsx 存在
ls components/CareCalculator.tsx

// 3. 測試
npm run dev
```
